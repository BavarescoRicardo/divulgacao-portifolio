## Installation

### Necessária a criação do arquivo .env com suas variavéis de ambiente 
  * DATABASE_CONECTION informar caminho da conexão com o Mongo DB atlas
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
